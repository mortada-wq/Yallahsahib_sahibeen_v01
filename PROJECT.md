# Yalla Sahib — Project Notes

Yalla Sahib is built and maintained by **Mortada Gzar**.

## How we organize work

Features are tracked as GitHub Issues. PRs are the unit of change — each PR description explains what changed and why, and becomes the changelog entry when merged.

## Priorities

1. Auth reliability (Firebase flows)
2. Provider compatibility (keeping all 10+ AI providers working)
3. Deploy integrations (Vercel, Netlify, Docker)
4. Desktop experience (Electron packaging, auto-update)

## Contributing

Open an issue to discuss before building large features. Small fixes can go straight to a PR.
